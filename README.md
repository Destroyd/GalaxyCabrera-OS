# 🌌 GalaxyCabrera OS

> **Open Media Platform**

Una plataforma multimedia moderna, modular y de código abierto para servidores domésticos, diseñada para integrarse con **CasaOS**, **Jellyfin**, **Plex**, **Docker** y **Cloudflare Tunnel**.

> **Estado del proyecto:** 🚧 En desarrollo (v0.1.0-alpha "Andromeda")

---

## 🎯 Objetivo

GalaxyCabrera OS busca ofrecer una experiencia multimedia moderna y elegante, inspirada en servicios de streaming profesionales, pero completamente autohospedada y de código abierto.

## Características previstas

- 🎬 Integración con Jellyfin
- 🎥 Integración con Plex
- 📦 Compatibilidad con CasaOS
- ☁️ Acceso seguro mediante Cloudflare Tunnel
- 🐳 Despliegue con Docker
- 🌌 Tema visual Galaxy Dark
- 📊 Dashboard del servidor
- ⚙️ Instalador modular
- 🔄 Actualizaciones sencillas
